# Deepfake & AI-Generated Image Detection

Detecting AI-generated (deepfake) images, centred on the **CIFAKE** dataset — an MSc Deep Learning project.

CIFAKE pairs the 60,000 real CIFAR-10 images with 60,000 synthetic images generated by **Stable Diffusion 1.4** (120k total, 32×32 RGB, 100k train / 20k test, balanced binary classification). This repo explores classical baselines, deep models trained from scratch, fine-tuned backbones, foundation-model embeddings, and — most importantly — the **cross-generator generalization gap** and **robustness** that separate a real detector from one that only memorizes a single generator.

## Goals

1. **Reproduce baselines** — the original CIFAKE CNN (~92.98%) and a handcrafted-feature + LightGBM classical baseline (~F1 0.94).
2. **Train deeper models** — an improved residual/dual-branch CNN from scratch, and fine-tuned backbones (ResNet50V2 / EfficientNet, a ViT with LoRA). Target ≥96%.
3. **Study generalization & robustness** — CLIP/DINOv2 linear probes, cross-generator evaluation (GenImage / Synthbuster), and robustness curves (JPEG / blur / noise).
4. **Explainability** — Grad-CAM, attention rollout, and frequency-spectrum visualizations (reproduce the CIFAKE "background, not object" finding).

See [docs/research.md](docs/research.md) for the full method survey and references.

## Getting started

```bash
# 1. Clone
git clone https://github.com/konstantinosmpouros/deepfake-detection.git
cd deepfake-detection

# 2. Create an environment (Python 3.10+)
python -m venv .venv
# Windows
.venv\Scripts\activate
# macOS / Linux
source .venv/bin/activate

# 3. Install dependencies
pip install -r requirements.txt
```

### Data

The CIFAKE dataset is **not** committed to the repo. Download it (e.g. from
[Kaggle](https://www.kaggle.com/datasets/birdy654/cifake-real-and-ai-generated-synthetic-images))
and place it under `data/cifake/` following the train/test layout.

## Stack

PyTorch · timm · Hugging Face transformers + peft · open_clip · OpenCV · scikit-image · scikit-learn · lightgbm / xgboost · matplotlib / seaborn · pytorch-grad-cam.

A single mid-range GPU (8–12 GB) is sufficient at 32×32; the classical baselines run on CPU.

## Team & scope

3-person project (~4–6 weeks):

- **Classical & EDA** — dataset EDA, frequency analysis (FFT/DCT, azimuthal power spectrum), handcrafted features (HOG/LBP/GLCM/DCT/wavelets) + LightGBM/SVM.
- **Deep models** — reproduce the CIFAKE CNN, train an improved residual/dual-branch CNN, fine-tune ResNet50V2/EfficientNet and a ViT (with LoRA).
- **Foundation models, generalization & robustness** — CLIP/DINOv2 linear probe, cross-generator eval, robustness curves, explainability.
